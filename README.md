# Bertrand's Box

A Simulation of Bertrand's Box

## Usage

    bertrands_box [ROUNDS]

see `bertrands_box -h` for more details.

## Installation

    python3 -m pip install git+https://github.com/therealzanfar/bertrands_box.git
